# ✨webpack-boilerplate

## Install

```
npm install
```



## Build

### production(배포)

```
npm run build
```

### development(개발)

```
npm run build:dev
```





## Dependency

- webpack
- style-loader css-loader sass-loader
- babel-loader babel-core babel-polyfill
- source-map
